HELP PAGE

here are some important things which you must know before you start creating new pages in your textbook/notebook.

Add page as Text/Image

If you want to upload your page as an image, then choose Page as Image otherwise choose Page as Text and you can write your page content.

Chapter Name

You can use this button to enter your Chapter Name (Only in case of Page As Text) .

Heading

You can use this button to enter your Headings in the page(Only in case of Page As Text).

Scientific Key

This button active only after clicking on start writing button,and you can enter scientific formulas by using this button(Only in case of Page As Text).

Add Image

By using this button you can upload images in your page(Only in case of Page As Text).

Add Page

By using this button you can upload your page in jpeg/png format (Only in case of Page As Image).

Help

By clicking this button you will be redirected to the help page

Submit

You can save your page by clicking on this button.
